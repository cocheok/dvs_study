Software

El desarrollo de software consiste tomar las librerias publicadas por Robotics perception Group adaptarlas a los flujos de Node-Red En el caso del nodo de ROS dvs_ros_driver se crea un nodo de Node-Red bajo la categoria DVS128

Dicha adaptación se realiza utilizando la libreria de NodeJS rosjs que es la que realiza la comunicación con el roscore a traves de rosbridge y manteniendo el prefijo node-red- como nombre del proyecto de NodeJS de npm realizado.
